Installation

To install Compy you need these things:
  • Python 2.7 running in your system (to check if you have it, open a terminal and type python, it should open a python console
  • A python library called WxPython, check if you have it typing import wx in the python console
  • Some other python libraries: jinja2,
  • The Compy program

To run compy, just open a terminal in the compy directory and type: 


    $ pythonw hello_world.py.

Important: If you don't know what I'm talking about, don't worry, just find someone who is more into computers and ask him or her to set things up for you
