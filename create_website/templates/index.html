The Compy Project - Index

The Compy Project is an personal project of a father of four, who learned programming with a Commodore Vic-20 and, much later, fell in love with python.

The Compy Project aims to blend the beautiful, colorful and easy screen familiar to Commodore users with the elegant syntax of python.

To start, check how to install it, then run the examples, and then, and only then, try the challenges.
